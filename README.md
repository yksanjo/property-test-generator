# Property-Based Test Generator

A tool for generating property-based tests using Hypothesis. Tests verify code properties across many random inputs.

## Features

- Hypothesis-based property testing
- Automatic strategy generation for types
- Consistency and inverse property tests
- Integration with pytest

## Usage

```bash
python property_test_generator.py --source /path/to/code --output ./tests
```

## Installation

```bash
pip install -r requirements.txt
pip install hypothesis
```
